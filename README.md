# Pyrolysis_regressor_grapp
### “Pyrolysis_regressor” - это приложение модели процесса пиролиза нефтяного сырья на основе нейронной сети/модели машинного обучения. Цель данного приложения - это прогноз, на основе ввёденных пользователем параметров процесса пиролиза, выходных целевых продуктов: этилена и пропилена.
#### Общий вид веб-приложения
![image](https://github.com/Ilfick/Pyrolysis_regressor_grapp/assets/113624229/850982c1-bbfd-46f4-b8d3-f4dd7db493b4)
#### Блок ввода данных пользователем. 
##### Входные параметры: начало времени работы печи пиролиза и время окончания; доли этана, пропана, пара, а также температура процесса пиролиза. По нажатию кнопки "Запустить печь пиролиза" будет осуществлён вывод спрогнозированных данных. Получить графики выхода целевых продуктов можно нажав кнопку «График выхода» 
![image](https://github.com/Ilfick/Pyrolysis_regressor_grapp/assets/113624229/6ab7b1ff-16d6-44a5-b5e5-f9eac1088499)
#### Блок интерфейса для сохранения данных в файл.
##### Пользователь может выбрать формат файла при помощи выпадающего списка: "csv" или "xls", далее ввести название файла в поле «Введите название файла» и сохранить получившийся файл при нажатии на кнопку «Сохранить данные в файл»
